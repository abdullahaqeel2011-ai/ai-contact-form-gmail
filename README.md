# AI Contact Form → Smart Reply → Gmail 🤖✉️

Automate your client communications and save time! This workflow demonstrates how to use **n8n** and **OpenAI** to automatically respond to client inquiries through Gmail.

## Features
- 📝 **Capture inquiries:** Collect client messages via a custom web form  
- 🤖 **Smart AI replies:** Generate professional, friendly email responses  
- ✉️ **Automated emails:** Send replies directly via Gmail  
- ⏱️ **Time saver:** Reduce manual work and stay responsive  

## How It Works
1. A client submits a message through the **AI Contact Form**  
2. n8n captures the form data and formats it  
3. OpenAI generates a professional reply  
4. The workflow sends the email automatically via Gmail  

## Benefits
- Maintain a professional client experience  
- Respond quickly and efficiently  
- Free your team from repetitive email tasks  
- Easily customizable for any business  

## Requirements
- n8n account  
- OpenAI API key  
- Gmail account with OAuth2 credentials  

## Installation
1. Import the workflow JSON into n8n  
2. Connect your OpenAI and Gmail credentials  
3. Customize the form and email templates  
4. Activate the workflow  

## Usage
Simply share the form with your clients. When they submit inquiries, AI-powered responses are sent automatically — no manual effort required!
